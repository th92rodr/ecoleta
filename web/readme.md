# Ecoleta SPA App (Web Frontend)

## Stack

- <a href="https://reactjs.org/"><img src="https://img.shields.io/badge/React.js-blue.svg?logo=react" alt="React.js"></a>

## Run Application

- Install project dependencies:

```
npm i
```

- Start app:

The app will be running on port `3000`.

```
npm start
```

## App Routes

- `/`:

Home page.

- `/create-point`:

Create point page.
